# Nazmus Sakib Ohee - Software Engineer Portfolio

Visit: https://sakibohee.github.io/

## Overview
This is a professional portfolio website for Nazmus Sakib Ohee, a Senior Software Engineer specializing in Laravel-based web development. The site showcases his experience, skills, projects, and achievements.

## Features
- **Responsive Design**: Built with Bootstrap for mobile-first responsive design
- **Dynamic Content**: All content is loaded from a JSON file (`data.json`) for easy maintenance
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Sections Include**:
  - Hero section with introduction
  - About section
  - Skills with progress bars
  - Achievements carousel
  - Experience and Education tabs
  - Services offered
  - Featured projects
  - Client testimonials
  - Contact information

## Structure
- `index.html`: Main HTML file with embedded JavaScript for dynamic content loading
- `data.json`: Contains all the website content in JSON format
- `css/`: Stylesheets (Bootstrap, custom styles)
- `js/`: JavaScript libraries and custom scripts
- `img/`: Images and logos
- `vendors/`: Third-party libraries (Owl Carousel, etc.)

## How to Update Content
1. Edit the `data.json` file to update any content
2. The website will automatically reflect changes when reloaded
3. No need to modify HTML or JavaScript for content updates

## Technologies Used
- HTML5

- JavaScript (jQuery)
- Bootstrap 4
- Owl Carousel
